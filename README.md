# Safe Steps – Road Safety Awareness Project

Safe Steps is a school-based road safety awareness campaign aimed at improving learner safety around Hoerskool Jan Viljoen. 

The project combines data collection, analysis, and practical solutions to address unsafe road behaviour and environmental risks.

## 🚀 Features

- Survey data collection and analysis
- Visual graphs showing key findings
- Problem analysis based on real data
- Structured solutions and action plan
- Fundraising strategy with cost and profit estimates
- Gallery showcasing campaign activities and team
- Contact page for communication and support

## 🛠️ Tech Stack

- HTML5
- CSS3
- Google Forms & Google Sheets (for data collection)
- Netlify (hosting)
